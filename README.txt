## Aperçu

Ce projet vise à créer une application de détection et de suivi d'objets en temps réel en utilisant ROS2 et OpenCV.
- Partie 1 : Détection et Suivi de Couleurs.
- Partie 2 : Reconnaissance d'Objets.

## Structure des Fichiers

- detection_couleur.py : Script Python pour la détection et le suivi de couleurs.
- detect.py : Script Python pour la détection et la reconnaissance d'objets.

## Dépendances

- ROS2
- OpenCV
- Python 3

## Notes

- Assurez-vous que votre environnement ROS2 est correctement configuré.
- Vérifiez que les bibliothèques OpenCV sont accessibles dans votre environnement Python.
